Project on Console


What is Visual Studio Code?
Visual Studio Code is an IDE (Integrated Development Environment). It is a source code editor used to build websites and applications.


What is the console screen?
The console screen is used to run code before finalizing on the main output screen. The Console Window displays errors, warnings, and other messages the Editor generates. These errors and warnings help you find issues in your project, such as script compilation errors.



To use the console screen:
1. Right click and open Insert.
2. Click "console" at the top.
3. Use code "console.log()" in the editor to display anything on the console screen
